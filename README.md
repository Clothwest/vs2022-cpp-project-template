# VS2022 Cpp Project Template

A template to generate Visual Studio 2022 project.

## How to Use

1. Modify `premake5.lua` ~~or not~~
2. With [premake5](https://github.com/premake/premake-core) on your computer, simply double click `generate.cmd` to generate solution and project
